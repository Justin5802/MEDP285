Op Art

This is a website dedicated to op art, which explores geometric abstraction with geometric shapes, such as with circles, squares, triangles, ovals, and rectangles.

What is on this Website?

This website features op art, also known as optical art, which is a visual style that uses geometric shapes and objects to create optical illusions and effects. This website features op art created using CSS.
